# 仪表盘改造方案（基于现有系统）

## 现状分析

**已有数据（可直接展示）：**
- 15 技能等级/经验
- 金币/骨头/龙骨/血碎晶/石头
- 当前正在执行的动作（activeAction）
- 家园建筑状态（homestead buildings）
- 战斗等级（combatLevel）
- 世界层级（worldTier）
- 离线产出（tick 系统自动计算，无需额外开发）

**没有的数据（需要新增系统）：**
- 人口系统
- 电力系统
- 辐射值
- 任务系统

## 改造策略

**不新增系统**，只改 UI 展示层。用现有数据重新组织仪表盘，让它像一个废土避难所的监控中心。
