## 2. 各区域功能说明

### TopStatusBar (40px) — 顶部生存状态栏

深色金属底板上排列关键生存数据，使用 CRT 荧光绿色数字显示。

| 元素 | 说明 |
|------|------|
| ⚡ 电力 | 避难所当前电力/最大电力，低于20%闪红 |
| ☢ 辐射 | 当前辐射值百分比，用黄→红渐变条 |
| 🌡 温度 | 避难所温度，低于10°C显示蓝色警告 |
| 废铁/木材/净水/口粮 | 四大基础资源，数字跳变动画 |

### LeftPanel (200px) — 技能监控面板

垂直排列15项技能，每项一行紧凑显示：
- 技能图标(24×24) + 名称 + 等级
- 微型进度条(80×4)
- 当前正在训练的技能够绿色荧光高亮
- 点击跳转到该技能详情页

### CenterPanel — 避难所全景（视觉核心）

这是整个界面的灵魂。不再是一张静态图片，而是有生命的基地：

**上部：3×3 避难所建筑网格**
- 9个建筑槽位，初始大部分为空
- 已建成的显示图标+等级（水电/食堂/医疗/仓库/工坊/哨塔/居住区）
- 空格闪烁提示"可建造"
- 建筑有工作动画（工坊冒烟、哨塔探照灯旋转）

**中部：当前行动仪表盘**
- CRT 终端风格显示正在执行的行动
- 模拟 Pip-Boy 的绿色荧光文字
- 实时进度条（辐射绿填充）
- 预计剩余时间
- 产出预览（本次已采集：铜丝矿 ×12，预计获得：×34）
- 大大的 [停止] 按钮（锈铁红）

**下部：离线收益面板（登录时弹出）**
- "你离开了 3 小时 42 分"
- 列表展示这期间所有技能的产出
- 一键领取按钮

### RightPanel (260px) — 任务与事件面板

**上部：任务面板**
- 主线任务（橙色高亮，不可删除）
- 每日任务（黄色，带倒计时）
- 可选任务（灰色）

**下部：世界事件**
- 随机事件（废土商人来访、掠夺者袭击、辐射风暴）
- 倒计时显示
- 彩色事件图标

## 3. 中央区域的详细设计

避难所全景是最重要的视觉核心，应该占据屏幕60%的面积。

### 3×3 建筑网格设计

```
┌──────────┬──────────┬──────────┐
│ ⚡ 发电机 │ 🍲 食堂  │ 🏥 医疗站│  第1行：生存设施
│  Lv.3    │  Lv.2    │  Lv.1    │
│ ⚡+12/h  │ 🍲+8/h  │ 💊+3/h  │
├──────────┼──────────┼──────────┤
│ 📦 仓库  │ 🔧 工坊  │ 🗼 哨塔  │  第2行：生产防御
│  Lv.2    │  Lv.3    │  [建造]  │
│ 📦500容 │ ⚙+5/h  │ 🛡 防御  │
├──────────┼──────────┼──────────┤
│ 🛏 居住A │ 🛏 居住B │ 🏗 [空]  │  第3行：扩张空间
│  Lv.1    │  Lv.1    │  可建造  │
│ 👥+2人口 │ 👥+2人口 │  需要石材│
└──────────┴──────────┴──────────┘
```

每个建筑格：
- 80×80 像素
- 锈铁边框（9-Slice）
- 建造完成显示写实图标+等级
- 未建造显示虚线框+资源需求提示
- 点击弹出升级/拆除面板
- 产出资源用绿色数字缓慢上飘

嵌在建筑网格下方，模拟 CRT 终端：

## 4. 如何减少留白

| 策略 | 实现 |
|------|------|
| 建筑网格填满中央 | 3×3 网格 + 当前行动面板占据中央80%面积 |
| 右侧任务/事件面板 | 始终有内容（空任务显示"暂无新任务，去探索废土吧"） |
| 背景纹理 | 深色金属拉丝纹理 + 锈斑 + 铆钉装饰 |
| 角落装饰 | 四角锈蚀金属框，顶部通风管道剪影 |
| 人物剪影 | 底部边缘有居民剪影在工作 |
| 浮动粒子 | 微小的灰尘/辐射粒子缓慢飘动 |

## 5. 废土氛围设计

### 配色方案

| 用途 | 颜色 | 说明 |
|------|------|------|
| 主背景 | #0D0D0A | 极深黑褐 |
| 面板背景 | #1A1A14 | 深灰棕 |
| 边框 | #3D2B1F | 锈铁棕 |
| 主文字 | #C4B5A5 | 暖灰白 |
| 荧光绿 | #4DFF4D | CRT 终端的生命色 |
| 警告红 | #D64045 | 危险/警告 |
| 黄铜 | #B8863C | 金属高光 |

### 视觉元素

- **扫描线**：所有面板覆盖 2px 间隔半透明横线（CRT 效果）
- **铆钉**：面板四角各一个 6×6 铆钉图
- **锈斑**：面板边缘随机分布的锈蚀纹理
- **荧光闪烁**：关键数字（电力/辐射）间歇性微微闪烁
- **背景音效**（可选）：低频嗡嗡声 + 偶尔金属碰撞声

### 字体

- 标题：Oswald / Impact（工业感厚重字体）
- 正文/数字：Courier New / Roboto Mono（终端等宽字体）
- 特殊：荧光绿文字用 `text-shadow: 0 0 4px #4DFF4D`

## 12. 实现优先级

| 优先级 | 内容 | 理由 |
|--------|------|------|
| P0 | 配色+字体全局改造 | 最快速出氛围 |
| P0 | 中央避难所建筑网格 | 视觉核心，定义游戏身份 |
| P0 | 顶部生存状态栏 | 即时反馈生存压力 |
| P1 | 左侧技能监控面板 | 已有数据，改造展示形式 |
| P1 | 当前行动 CRT 终端面板 | 让"正在执行"可见 |
| P1 | 底部快捷栏 | 导航核心 |
| P2 | 右侧任务/事件面板 | 需要新增数据支持 |
| P2 | 离线收益弹窗 | 已有离线产出逻辑 |
| P3 | 扫描线/粒子/动画 | 氛围打磨 |
